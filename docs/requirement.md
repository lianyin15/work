# 需求说明

## 项目名称

学习打卡与成就激励系统

## 项目简介

轻量级学习打卡平台，用户可创建打卡任务、每日打卡获取积分，系统根据连续打卡天数和累计次数自动发放成就徽章，并提供积分排行榜。

## 用户角色

| 角色 | 说明 |
|---|---|
| 普通用户 | 注册登录后创建任务、每日打卡、查看排行与徽章 |

## 核心功能

| 功能 | 说明 |
|---|---|
| 用户注册/登录 | 支持用户名密码注册和登录，JWT 鉴权 |
| 任务创建 | 用户创建打卡任务，设置标题、描述、起止日期 |
| 每日打卡 | 对任务进行每日打卡，系统自动计算连续天数和积分 |
| 积分计算 | 连续第 n 天打卡获得 n 积分，积分累计到用户账户 |
| 成就徽章 | 首次打卡、连续 7 天、连续 30 天、累计 50 次、累计 100 次自动发放徽章 |
| 排行榜 | 按总积分展示全用户排名（前 50） |
| 个人中心 | 查看个人信息、总积分、已获得的徽章 |

## 约束条件

- 前后端分离。
- 本地启动。
- 使用 MySQL 数据库持久化。
- 前端 6 个页面（登录、注册、任务列表、创建任务、任务详情、排行榜、个人中心）。
- 后端 10 个 API 接口。
- 数据库 5 张表（users, tasks, checkins, badges, user_badges）。
