# 后端项目

## 说明

此目录放置后端项目代码。

你可以选择任意后端技术：
- Spring Boot（推荐 Java）
- Express / NestJS（Node.js）
- Flask / FastAPI（Python）
- Gin（Go）
- 其他你喜欢的技术

## 启动方式

> 在此处写清你的后端项目的启动命令。

示例（使用 Spring Boot + Maven）：
```bash
cd backend
mvn spring-boot:run
```

## 接口清单

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | /api/xxx | 查询列表 |
| POST | /api/xxx | 新增记录 |
| GET | /api/xxx/{id} | 查询详情 |
| PUT | /api/xxx/{id} | 更新记录 |
| DELETE | /api/xxx/{id} | 删除记录 |
